# xdm67x

Personal portfolio website built with Rust, Leptos, and Trunk.

## Tech Stack

- [Leptos](https://leptos.dev/) (CSR mode)
- Stylance (CSS Modules)
- Trunk (build tool)

## Development

```bash
trunk serve
```

## Deployment

Deployed to GitHub Pages at [xdm67x.github.io](https://xdm67x.github.io)

Automatically deployed on push to `main` branch via GitHub Actions workflow.
